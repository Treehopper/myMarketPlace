myMarketPlace
=============

Composite p2 repository for Eclipse plug-in that I choose from for almost every new Eclipse install.


How to use?
=============

- Fork
- Clone to $ANY_DIR
- Eclipse->Help->Install New Software...->Add...->Local...
- Navigate to $ANY_DIR/myMarketPlace/repository
